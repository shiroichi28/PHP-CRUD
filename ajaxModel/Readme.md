
# AJAX MODAL CRUD 


AJAX Modal CRUD is a technique that allows for creating, reading, updating, and deleting (CRUD) data using AJAX requests within a modal window.

## Tech Stack

- **Client:** HTML, BootStrap 5

- **Server:** PHP

- **Database:** MySQL

- **Functionalities:** Jquery,AJAX

- **Tools:** XAMPP, VSCode
